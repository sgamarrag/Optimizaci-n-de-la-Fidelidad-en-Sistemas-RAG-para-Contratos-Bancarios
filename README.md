📜 Legal-RAG: Asistente Inteligente para Auditoría de Contratos HipotecariosEste proyecto implementa un sistema de Generación Aumentada por Recuperación (RAG) de alta fidelidad, diseñado específicamente para la interpretación y consulta de contratos bancarios y legales complejos. El objetivo principal es minimizar alucinaciones y maximizar la precisión mediante una arquitectura de recuperación en dos etapas.🚀 Características PrincipalesArquitectura de Embudo (Two-Stage Retrieval): Combinación de búsqueda híbrida (BM25 + FAISS) con un paso de Re-ranking semántico.Detección de Alucinaciones: Implementación de métricas de Grounding y ROUGE-L para validar la fidelidad de la respuesta frente al documento original.SLM Optimizado: Uso de Qwen 2.5 (1.5B) en formato ChatML para una generación determinista y eficiente en recursos (Google Colab compatible).Citación Automática: El sistema vincula cada afirmación con el fragmento específico del contrato original [n].🛠️ Stack TecnológicoLLM: Qwen/Qwen2.5-1.5B-InstructEmbeddings: paraphrase-multilingual-MiniLM-L12-v2Reranker: BAAI/bge-reranker-v2-m3 (Cross-Encoder)Vector Store: FAISS (HNSW Index con Normalización L2)Métricas: ROUGE-L & Grounding Score (custom)📐 ArquitecturaEl sistema procesa la información en tres fases críticas:Ingesta y Chunking: Limpieza de texto legal mediante Regex y fragmentación con solapamiento (overlap) para preservar el contexto de las cláusulas.Recuperación Híbrida: Fusión de resultados léxicos y semánticos mediante RRF (Reciprocal Rank Fusion).Filtrado por Reranker: Evaluación profunda de relevancia entre la pregunta del usuario y los candidatos recuperados, descartando ruido con un umbral de confianza.📊 Resultados de EvaluaciónTras el proceso de optimización, el sistema alcanzó los siguientes niveles de desempeño:MétricaResultado FinalDescripciónROUGE-L1.00Fidelidad semántica total frente a respuestas de referencia.Grounding Score1.00Ausencia de alucinaciones; toda la información está anclada al contrato.💻 Instalación y UsoClona el repositorio:Bashgit clone https://github.com/tu-usuario/legal-rag.git
